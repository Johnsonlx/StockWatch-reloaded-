Ohne Sync funktioniert StockWatch nur auf diesem Gerät. Mit Supabase-Konto können Sie sich auf beliebig vielen Geräten anmelden und Ihr Portfolio ist überall synchron.

1
Kostenloses Projekt erstellen auf supabase.com
2
In Authentication → Settings: „Confirm email" deaktivieren (sonst Email-Bestätigung nötig)
3
In Database → Replication: Tabelle sw_portfolio aktivieren (für Echtzeit-Sync)
4
Im SQL Editor folgendes ausführen:
create table public.sw_portfolio ( id text primary key, user_id uuid not null references auth.users on delete cascade, symbol text not null, target_price float8 not null, name text default '', currency text default '', purchases jsonb not null default '[]'::jsonb, created_at timestamptz default now() ); alter table public.sw_portfolio enable row level security; create policy "own" on public.sw_portfolio for all using (auth.uid() = user_id) with check (auth.uid() = user_id);
4
